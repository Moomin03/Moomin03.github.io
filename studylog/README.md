**STUDYLOG**는 제가 공부하면서 이해한 내용을 작성한 글입니다.
잘못되어있는 내용이 있을 수 있지만, 올바른 내용을 전달하기 위해서 여러 검증단계를 거친다는 것만 알아주셨으면 좋겠습니다. 
더 좋은 글을 쓰기 위해, 도움이 되는 글을 쓰려고 노력하겠습니다.


```sh
다음은 디렉토리 구성입니다.
├─STUDYLOG
│  └─DEEPL
│  └─MACHINEL
└────README.md
```
